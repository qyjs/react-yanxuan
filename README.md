# react-yanxuan
模仿网易严选的简易react单页面应用

### 基于技术
- react15.6
- react-router4
- redux
- fetch
- webpack
- sass
- flexible(移动端适配)

### 使用方法
克隆到本地
```bash
npm install
// 开发
npm run dev
// 打包
npm run build
```
